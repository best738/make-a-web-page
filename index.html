Important Concepts

Lession 1: The Basics of the Web and HTML

The Web and HTML

The web is computers that talk to one another. For example, your computer talking to other computers called servers. It's' essentially a collection of HTML (Hypertext Markup Language) documents. Your computer browser makes requests to servers via a protocol called HTTP (>Hypertext Transfer Protocol).This video discusses Web components. HTML contains text (what you see) and markup (how it looks).

Tags and Elements

HTML documents are made up of elements, which consist of an opening and closing tag. Examples of tags and elements can be seen in my code in making this document.Check out this video.

Computers are Stupid

Computers are very literal and small mistakes can cause major problems. Writing in the correct manner is called correct syntax. By talking to a computer correctly you can tell it anything you want.

Inline and Block Elements

Block elements create an invisible box around content. The p (paragraph) tag/element is an example of a block element. The br (break) tag/element is an example of an inline element.

Lesson 2: Creating a Structured Document with HTML

Developer Tools

Block elements form invisible boxes with contents inside of them. Developer tools allows the user to see the structure the webpage in the form of boxes.

Document Object Model

Document Object Model (DOM) is the tree like structure of HTML. HTML elements can have other elements inside of them! In other words, boxes can have boxes upon boxes within them.

Indented HTML and Boxes

The more indented an element is the more deeply nested it is in its corresponding box. I can see this in how sublime "manages" the code. Sublime also creates vertical lines to help me line up the opening and closing of elements. Sublime is one of many text editors.

Lesson 3: CSS Style and HTML Structure

Redundancy

Redundancy is problamatic in that it leads to errors because changes have to be made in multiple places. An example is applying style (CSS) to a webpage. By apply divs and assigning classes, styling can be applied in one location.

CSS

CSS stands for Cascading Style Sheets. CSS gives the HTML elements style like borders and padding. CSS can be added by writing it into the HTML (either in the head or inline) or linking the HTML to a separate CSS file. Writing it inline is generally a bad idea due to redunancy issues that cause errors.
